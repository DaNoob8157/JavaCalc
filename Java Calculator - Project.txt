Rubric & Overview
🧮 Java GUI Calculator Project Rubric
📌 Project Overview
In this project, students will build a Java Calculator Application using:
* Java Swing (GUI)

* Layout Managers

* ActionListeners

* Object-Oriented Programming (OOP)

Students will progress through three levels:
   * Core (Required) → Functional calculator

   * Intermediate → Additional tools and features

   * Advanced → Complex logic and architecture

Student CheckList

________________


🧮 Calculator Project Checklist
✅ CORE (Required)
         * I created a calculator GUI with buttons

         * I used a Layout Manager (NOT null layout)

         * My display cannot be typed into freely

         * Buttons update the display correctly

         * I used ActionListeners

         * My calculator performs +, -, *, /

         * My program handles division by zero

         * I used at least 2 classes (GUI + Logic)

________________


🚀 INTERMEDIATE
            * I added multiple views (Calculator, Converter, Tip)

            * I implemented a calculation history

            * I built a converter (at least 5 types)

            * I created a tip calculator

________________


🧠 ADVANCED
               * My calculator follows order of operations

               * I added scientific functions

               * I improved architecture (MVC or JavaFX)

________________


🧹 CODE QUALITY
                  * My variable names are clear

                  * My code is organized into methods/classes

                  * My GUI and logic are separate

                  * My code is formatted properly

                  * I included helpful comments
